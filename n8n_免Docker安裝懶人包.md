## 課前準備檢查清單

開課前請完成以下事項：

- [ ] 確認電腦可連接網路

- [ ] 無法安裝者請提前告知（會安排備援方案）

## Windows 電腦安裝教學

### Step 1：安裝 Node.js（約 5 分鐘）

#### 1.1 下載安裝檔
1. 開啟瀏覽器，前往：**https://nodejs.org/zh-tw/**
2. 點擊綠色的 **「LTS」** 按鈕下載（請下載左邊的 LTS 版本，不要下載右邊的 Current 版）
3. 下載的檔案名稱類似：`node-vXX.x.x-x64.msi`（XX 為版本號，可能是 22、24 等）

> **LTS 是什麼？**
> LTS = Long Term Support（長期支援版），意思是最穩定、最推薦使用的版本。版本號會隨時間更新，只要下載頁面上標示「LTS」的就對了。

#### 1.2 執行安裝
1. 找到下載的 `.msi` 檔案（通常在「下載」資料夾）
2. 雙擊開啟，出現安裝精靈
3. 一直按「下一步（Next）」到底
   - 不需要更改任何設定，保持預設即可
4. 看到「Finish」表示完成！

#### 1.3 驗證安裝
1. 按鍵盤 `Win + R`，輸入 `cmd`，按 Enter
2. 黑色視窗（命令提示字元）打開後，輸入：
   ```
   node --version
   ```
3. 如果看到版本號（格式如 `v22.x.x` 或 `v24.x.x`），代表成功！

**如果顯示「不是內部或外部命令」：**
- 請重新開機後再試一次
- 若仍不行，請在 LINE 群組求助

---

### Step 2：安裝 n8n（約 3-5 分鐘）

#### 2.1 開啟命令提示字元
- 按 `Win + R`，輸入 `cmd`，按 Enter

#### 2.2 輸入安裝指令
在黑色視窗中，**複製並貼上**以下指令（在視窗中按右鍵即可貼上）：

```bash
npm install n8n -g
```

#### 2.3 等待安裝完成
- 會看到很多文字在跑，這是正常的
- 可能需要 3-10 分鐘（依網路速度）
- 等到出現 `added X packages in Xs` 即可
- **安裝過程中請不要關閉視窗！**

**如果出現權限錯誤：**
- 請關閉 cmd，重新以「系統管理員身分」開啟
- 方法：開始選單搜尋「cmd」→ 右鍵 → 以系統管理員身分執行

---

### Step 3：啟動 n8n（每次上課都這樣做）

#### 3.1 開啟命令提示字元
- 按 `Win + R`，輸入 `cmd`，按 Enter

#### 3.2 輸入啟動指令
```bash
n8n start
```

#### 3.3 等待啟動完成
- 會看到文字不斷滾動
- 等待出現以下訊息：
  ```
  Editor is now accessible via:
  http://localhost:5678
  ```
- 這代表 n8n 已成功啟動！

#### 3.4 開啟瀏覽器使用
1. 打開 Chrome 或 Edge 瀏覽器
2. 網址列輸入：`http://localhost:5678`
3. 看到 n8n 的歡迎畫面，就代表成功了！

**第一次啟動會要求設定：**
- Email：輸入你的信箱
- Password：設定密碼（**請記住這組密碼！建議寫在筆記本上**）
- First Name：你的名字
- Last Name：你的姓氏

---

### Step 4：關閉 n8n

上課結束或需要關閉時：

1. 回到黑色命令視窗（不要關閉瀏覽器）
2. 按 `Ctrl + C`（按一次）
3. 會問 `Terminate batch job (Y/N)?`，輸入 `Y`，按 Enter
4. 視窗關閉即完全停止

**小提醒：**
- 關閉命令視窗 = n8n 停止運作
- 下次上課需要重新執行 `n8n start`

## Mac 電腦安裝教學

### Step 1：安裝 Node.js（約 5 分鐘）

#### 1.1 下載安裝檔
1. 開啟瀏覽器，前往：**https://nodejs.org/zh-tw/**
2. 點擊綠色的 **「LTS」** 按鈕下載
3. 下載的檔案名稱類似：`node-vXX.x.x.pkg`

#### 1.2 執行安裝
1. 雙擊下載的 `.pkg` 檔案
2. 安裝精靈開啟，一直按「繼續」
3. 會要求輸入電腦密碼（安裝軟體需要權限）
4. 看到「安裝完成」即可

#### 1.3 驗證安裝
1. 打開「終端機」（Terminal）
   - 方法：按 `Cmd + 空白鍵`，輸入 `terminal`，按 Enter
2. 輸入：
   ```
   node --version
   ```
3. 看到版本號（格式如 `v22.x.x` 或 `v24.x.x`），代表成功！

---

### Step 2：安裝 n8n（約 3-5 分鐘）

#### 2.1 開啟終端機
- 按 `Cmd + 空白鍵`，輸入 `terminal`，按 Enter

#### 2.2 輸入安裝指令
```bash
sudo npm install n8n -g
```

#### 2.3 輸入電腦密碼
- 輸入密碼時**不會顯示任何字元**（這是正常的安全措施）
- 輸入完按 Enter

#### 2.4 等待安裝完成
- 出現 `added X packages` 即可

---

### Step 3：啟動 n8n（每次上課都這樣做）

#### 3.1 開啟終端機

#### 3.2 輸入啟動指令
```bash
n8n start
```

#### 3.3 等待啟動完成
- 看到 `Editor is now accessible via: http://localhost:5678` 即可

#### 3.4 開啟瀏覽器
- 打開 Safari 或 Chrome
- 網址：`http://localhost:5678`

---

### Step 4：關閉 n8n

1. 回到終端機視窗
2. 按 `Ctrl + C` 即可停止（Mac 不需要額外確認）

## 常見問題排除（FAQ）

### Q1：輸入指令顯示「'node' 不是內部或外部命令」

**原因**：Node.js 沒有安裝成功

**解決方法**：
1. 重新開機後再試一次
2. 檢查 Node.js 是否安裝成功（控制台 → 程式和功能）
3. 若仍不行，請洽課堂助教協助

---

### Q2：安裝 n8n 時卡在原地很久（超過 10 分鐘）

**原因**：網路較慢或 npm 暫存問題

**解決方法**：
1. 耐心等待，可能需要 10-15 分鐘
2. 不要關閉視窗，讓它繼續跑
3. 若真的卡住，按 `Ctrl + C` 取消，再重新執行安裝指令
4. 如果在學校/公司網路，可能有網路代理（proxy）問題，請洽助教

---

### Q3：瀏覽器開啟 localhost:5678 顯示「無法連線」

**原因**：n8n 還沒啟動完成，或輸入錯誤

**解決方法**：
1. 回到命令視窗，確認有看到 `Editor is now accessible`
2. 如果沒有，重新輸入 `n8n start`
3. 確認網址是 `http://localhost:5678`（不是 https）
4. 確認沒有打錯字（是 5678，不是 5768 或其他）

---

### Q4：畫面顯示「There was a problem loading the page」或一直轉圈

**原因**：第一次啟動需要初始化資料庫

**解決方法**：
1. 等待 1-2 分鐘（第一次會比較慢）
2. 按 `F5` 或 `Ctrl + R` 重新整理頁面
3. 如果還是不行，關閉 n8n（Ctrl + C）再重新啟動

---

### Q5：忘記 n8n 的登入密碼

**解決方法**：
1. 關閉 n8n（Ctrl + C）
2. 刪除設定檔後重新啟動（會重建新帳號）
   - Windows：刪除 `C:\Users\你的使用者名稱\.n8n` 資料夾
   - Mac：刪除 `~/.n8n` 資料夾
3. 重新執行 `n8n start`，建立新帳號

---

### Q6：電腦記憶體不足（4GB 以下），跑得很慢

**建議**：
1. 上課時關閉其他不必要的程式（關閉 Chrome 其他分頁、關閉 LINE 電腦版、關閉音樂軟體等）
2. 若真的無法運作，請告知助教安排備援方案

---

### Q7：公司/學校電腦無法安裝（權限不足）

**建議**：
- 請使用個人筆電
- 或洽助教安排使用課堂共用主機

---

### Q8：防毒軟體或防火牆阻擋安裝

**原因**：部分防毒軟體會阻擋 npm 全域安裝或 localhost 連線

**解決方法**：
1. 暫時關閉防毒軟體的即時防護（安裝完再開啟）
2. 將 Node.js 和 n8n 加入防毒軟體的白名單
3. 確認 Windows 防火牆允許 Node.js 的網路存取（第一次啟動時 Windows 會跳出詢問，請按「允許」）
4. 如果學校/公司有網路管理，可能需要設定 npm proxy：
   ```bash
   npm config set proxy http://proxy.school.edu.tw:8080
   npm config set https-proxy http://proxy.school.edu.tw:8080
   ```
   （網址請向學校 IT 人員確認）

---

### Q9：安裝完 n8n 後輸入 `n8n start` 顯示找不到指令

**原因**：npm 全域安裝路徑沒有加入系統環境變數

**解決方法（Windows）**：
1. 按 `Win + R`，輸入 `cmd`，按 Enter
2. 輸入以下指令找到 npm 全域路徑：
   ```bash
   npm config get prefix
   ```
3. 複製顯示的路徑（例如 `C:\Users\你的名稱\AppData\Roaming\npm`）
4. 搜尋「環境變數」→ 點擊「編輯系統環境變數」→ 環境變數 → Path → 新增 → 貼上路徑
5. 重新開啟 cmd 再試

**解決方法（Mac）**：
1. 在終端機輸入：
   ```bash
   export PATH="$(npm config get prefix)/bin:$PATH"
   ```
2. 若要永久生效，將上面那行加入 `~/.zshrc` 檔案

## 無法安裝的備案方案

如果您的電腦真的無法安裝，請不要擔心！課堂提供以下協助：

### 方案 A：兩人一組（推薦）
- 與另一位同學共用一台電腦
- 輪流操作，互相學習討論
- 實際上這樣學習效果往往更好！

### 方案 B：課堂共用主機
- 使用瀏覽器連線到課堂主機
- **只要電腦能上網、能開瀏覽器就能操作**
- 資料會保存在主機，下次上課繼續使用
- 適合：電腦太舊、無法安裝軟體、Mac 舊版本等

### 方案 C：課後補課
- 安排時間到教室使用公用電腦練習
- 或提供錄影讓您在家跟做

**如果無法安裝，請在 LINE 群組回報您的問題，我們會提前安排協助。**
