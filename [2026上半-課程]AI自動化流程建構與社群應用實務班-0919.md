| **計畫緣由** | 在數位經濟浪潮與AI技術的雙重驅動下，產業對人才的需求已從傳統的行銷操作，轉向具備**系統化思維**與**自動化流程建構能力**的複合 型數位專才。過去企業仰賴大量人力執行重複性的社群發文、數據整理與客戶互動；如今，市場更需要能**設計、建構並優化自動化工作流**，以應對快速變化的商業環境，提升營運效率的人才。 然而，目前多數課程仍偏重於社群行銷的「策略」與「內容」，對於如何運用Low-code/No-code工具串接服務、整合AI模型，以建構高效的自動化系統，則著墨甚少。這造成了明顯的人才斷層：企業找不到能解決流程效率問題的專才，求職者則缺乏能突顯其技術價值的關鍵技能。 本訓練計畫正為填補此缺口而生，旨在培養學員成為「**AI自動化流程架構師**」。我們將從根本上轉變學員的思維，從「執行者」提升為「**系統設計者**」，使其不僅懂得社群應用，更能**掌握建構自動化流程的核心技術**，成為推動企業數位轉型、提升競爭力的關鍵力量。 |  |  |
| 目標 | 本課程旨在培養學員成為兼具**自動化流程建構能力**與**社群應用策略**的專業技術人才。學員將透過本課程達成以下目標： **掌握自動化流程建構核心技術：** 熟練運用主流 Low-code/No-code 自動化平台 (如 n8n)，學習節點操作、數據流處理與邏輯判斷。 學習 API 串接實務，能整合社群平台、雲端服務與 AI 模型，建構跨系統的自動化工作流。 **整合 AI 模型於商業應用：** 學習如何將大型語言模型 (LLM) 作為自動化流程的智慧核心，實現內容自動生成、數據分析與智慧決策。 透過實作，將 AI 應用從單點工具操作，提升至系統性的流程優化層次。 **具備解決商業問題的實戰能力：** 透過專案導向的學習，學員將能獨立分析商業需求（如：社群內容管理、輿情監測、客戶關係維護），並設計與建構對應的自動化解決方案。 結訓時，每位學員將完成一個可展示的自動化專案作品集，證明其具備導入與優化企業數位流程的即戰力。 **輔導就業與專業認證：** 本課程的就業展望將聚焦於「**數位轉型工程師**」、「**營運自動化專員**」、「**AI流程架構師**」等新興技術職位。 輔導學員考取 **Google Analytics (GA4)** 等數據分析相關證照，強化其在數據判讀與成效追蹤上的專業能力，使其成為業界炙手可熱的數位資訊專才。 |  |  |
