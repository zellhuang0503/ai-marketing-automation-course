# 專案一：智能貼文小幫手 - 完整實作指南

## 專案概述

**專案名稱**：智能貼文小幫手  
**難度等級**：⭐⭐（基礎）  
**預估時間**：10小時  
**核心技術**：n8n AI Agent + LINE Bot + OpenAI API + Google Sheets

### 功能描述
```
使用者（LINE）
    ↓
輸入：「幫我寫一篇咖啡廳開幕的貼文」
    ↓
AI Agent 處理：
├── 判斷適合平台（FB/IG）
├── 生成 3 種風格文案（正式/輕鬆/趣味）
├── 建議最佳發文時間
└── 生成 Hashtag
    ↓
自動儲存到 Google Sheets
    ↓
發送完成通知到 LINE
```

## 第一部分：環境設定（2小時）

### Step 1：建立 LINE Bot 帳號

#### 1.1 登入 LINE Developers
1. 前往 https://developers.line.biz/
2. 點擊「Log in to Console」
3. 使用 LINE 帳號登入

【截圖位置：LINE Developers 登入頁面，標註 Log in 按鈕】

#### 1.2 建立 Provider
1. 登入後，點擊「Create New Provider」
2. Provider Name 輸入：`AI行銷課程練習`
3. 點擊「Create」

【截圖位置：Create New Provider 彈窗，填寫名稱處】

#### 1.3 建立 Messaging API Channel
1. 點擊剛建立的 Provider
2. 點擊「Create a Messaging API channel」
3. 填寫基本資訊：
   - Channel type：Messaging API
   - Channel name：`智能貼文小幫手`
   - Channel description：`AI自動生成社群貼文`
   - Category：其他
   - Subcategory：其他
4. 勾選同意條款，點擊「Create」

【截圖位置：填寫 Channel 資訊頁面】

#### 1.4 取得重要憑證
建立完成後，進入 Channel 設定頁面：

**取得 Channel Access Token：**
1. 左側選單點擊「Messaging API」
2. 找到「Channel access token」區塊
3. 點擊「Issue」產生 Token
4. **複製這串文字並妥善保存**（很長的一串）

【截圖位置：Channel access token 區塊，標註 Issue 按鈕和 Token 位置】

**取得 Channel Secret：**
1. 左側選單點擊「Basic settings」
2. 找到「Channel secret」
3. 點擊「Show」顯示
4. **複製這串文字並妥善保存**

【截圖位置：Channel secret 區塊】

**記下你的 Bot ID：**
- 在 Basic settings 頁面，找到「Your user ID」
- 格式類似：`Uxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

#### 1.5 將 Bot 加入好友
1. 在 Messaging API 頁面，找到「QR code」
2. 用 LINE App 掃描加入好友
3. 發送一則測試訊息給 Bot

【截圖位置：QR code 區塊】

---

### Step 2：取得 OpenAI API Key

#### 2.1 註冊 OpenAI 帳號
1. 前往 https://platform.openai.com/
2. 點擊「Sign up」註冊（可用 Google/微軟帳號）

#### 2.2 建立 API Key
1. 登入後，點擊右上角個人頭像 → 「API keys」
2. 點擊「Create new secret key」
3. 輸入名稱：`n8n專案一`
4. 點擊「Create secret key」
5. **立即複製這串 Key**（只會顯示一次！）

【截圖位置：API keys 頁面，標註 Create 按鈕】

> ⚠️ **重要提醒**：這個 Key 只會顯示一次，請務必保存好！

#### 2.3 確認額度
1. 點擊左側「Billing」→「Overview」
2. 確認還有可用額度（免費帳號通常有 $5 試用額度）

【截圖位置：Billing Overview 頁面，顯示可用額度】

---

### Step 3：建立 Google Sheets

#### 3.1 建立試算表
1. 前往 https://sheets.google.com/
2. 點擊「+ 空白」建立新試算表
3. 標題命名為：`智能貼文小幫手_紀錄`

#### 3.2 設定欄位標題
在 A1 儲存格開始，輸入以下欄位：
```
A1: 時間
B1: 使用者ID
C1: 輸入內容
D1: 平台
E1: 風格1_正式
F1: 風格2_輕鬆
G1: 風格3_趣味
H1: Hashtag
I1: 建議發文時間
```

【截圖位置：試算表欄位設定完成畫面】

#### 3.3 設定共用權限
1. 點擊右上角「共用」
2. 點擊「變更為任何知道連結的人」
3. 權限設為「編輯者」
4. 點擊「完成」

【截圖位置：共用設定視窗】

#### 3.4 記下試算表 ID
看網址列，格式如下：
```
https://docs.google.com/spreadsheets/d/[這裡是ID]/edit
```
複製 `d/` 和 `/edit` 中間的那串文字，這就是試算表 ID

### Step 4：建立 Google Cloud 專案（為了 Sheets API）

#### 4.1 建立專案
1. 前往 https://console.cloud.google.com/
2. 登入 Google 帳號
3. 點擊上方專案下拉選單 → 「New Project」
4. Project name：`n8n-automation`
5. 點擊「Create」

【截圖位置：New Project 視窗】

#### 4.2 啟用 Google Sheets API
1. 左側選單點擊「APIs & Services」→「Library」
2. 搜尋框輸入：`Google Sheets API`
3. 點擊「Google Sheets API」
4. 點擊「Enable」

【截圖位置：API Library 搜尋結果，標註 Enable 按鈕】

#### 4.3 建立服務帳號
1. 左側選單點擊「IAM & Admin」→「Service Accounts」
2. 點擊「Create Service Account」
3. Service account name：`n8n-sheets`
4. 點擊「Create and Continue」
5. Role 選擇：「Editor」（編輯者）
6. 點擊「Continue」→「Done」

【截圖位置：建立服務帳號流程】

#### 4.4 建立並下載金鑰
1. 在服務帳號列表中，點擊剛建立的 `n8n-sheets`
2. 點擊「Keys」頁籤
3. 點擊「Add Key」→「Create new key」
4. 選擇「JSON」格式
5. 點擊「Create」
6. 會自動下載一個 `.json` 檔案，**妥善保存這個檔案！**

【截圖位置：建立金鑰選項，標註 JSON】

#### 4.5 分享試算表給服務帳號
1. 回到 Google Sheets 試算表
2. 點擊「共用」
3. 加入服務帳號 email（在剛下載的 JSON 檔案中，尋找 `"client_email"` 欄位）
4. 權限設為「編輯者」
5. 點擊「傳送」

【截圖位置：共用視窗，顯示已加入服務帳號】

## 第二部分：建立 n8n 工作流程（6小時）

### Step 5：啟動 n8n 並建立新 Workflow

#### 5.1 啟動 n8n
```bash
n8n start
```

等待看到 `Editor is now accessible via: http://localhost:5678`

1. 瀏覽器開啟 `http://localhost:5678`
2. 點擊左側「Workflows」
3. 點擊「Add Workflow」
4. 點擊 workflow 名稱（預設是「My workflow」），改名為：`專案一_智能貼文小幫手`

【截圖位置：Workflows 列表頁面，標註 Add Workflow】

### Step 6：設定 LINE Trigger（觸發器）

#### 6.1 添加 LINE Trigger 節點
1. 在空白畫面中，點擊「+」或「Add first step」
2. 搜尋「LINE」
3. 選擇「LINE」→「On Message」
4. 點擊「Continue」

【截圖位置：選擇 Trigger 節點畫面】

#### 6.2 建立 LINE 連線（Credentials）
1. 在「Credential」下拉選單，選擇「Create New Credential」
2. 輸入名稱：`LINE_Bot_專案一`
3. 填寫：
   - Channel Access Token：貼上 Step 1.4 取得的 Token
   - Channel Secret：貼上 Step 1.4 取得的 Secret
4. 點擊「Save」

【截圖位置：LINE Credential 設定視窗】

#### 6.3 設定 Trigger
- Event Type：`Message`
- Message Type：`Text`

點擊「Test trigger」，然後在 LINE 發送一則訊息給 Bot，應該會看到測試資料出現。

【截圖位置：Trigger 設定完成，顯示測試成功】

---

### Step 7：設定 AI Agent 節點

#### 7.1 添加 AI Agent 節點
1. 點擊 LINE Trigger 右側的「+」
2. 搜尋「Agent」
3. 選擇「AI Agent」
4. 點擊「Continue」

【截圖位置：選擇 AI Agent 節點】

#### 7.2 設定 AI Agent 屬性

**基本設定：**
- Mode：`Run Once for All Items`
- Prompt：輸入以下內容：

```
你是一位專業的社群行銷文案助手。

用戶會輸入一個主題或情境，請幫他生成社群貼文文案。

請依照以下格式回覆：

📱 **建議平台**：[Facebook / Instagram]

📝 **文案草稿**：

【正式風格】
[這裡放正式風格的文案]

【輕鬆風格】
[這裡放輕鬆風格的文案]

【趣味風格】
[這裡放趣味風格的文案]

#️⃣ **建議Hashtag**：
[提供5-8個相關Hashtag]

⏰ **建議發文時間**：
[建議的最佳發文時間]

---

用戶輸入：{{ $json.message.text }}
```

【截圖位置：AI Agent Prompt 設定畫面】

**連結 OpenAI：**
1. 在「Options」區塊，找到「Language Model」
2. 選擇「OpenAI」→「Create New Credential」
3. 輸入名稱：`OpenAI_專案一`
4. API Key：貼上 Step 2.2 取得的 Key
5. 點擊「Save」

【截圖位置：OpenAI Credential 設定】

**選擇模型：**
- Model：`gpt-3.5-turbo`（或 `gpt-4` 如果你有權限）
- Temperature：`0.7`（創意程度，0.7 是平衡值）

【截圖位置：AI Agent 完整設定畫面】

#### 7.3 測試 AI Agent
點擊「Test step」，應該會看到 AI 回應的內容。

【截圖位置：AI Agent 測試成功，顯示回應內容】

### Step 8：設定 Google Sheets 節點（記錄功能）

#### 8.1 添加 Google Sheets 節點
1. 在 AI Agent 右側點擊「+」
2. 搜尋「Google Sheets」
3. 選擇「Google Sheets」→「Append Row in Sheet」
4. 點擊「Continue」

【截圖位置：選擇 Google Sheets 節點】

#### 8.2 建立 Google Sheets 連線
1. 在「Credential」選擇「Create New Credential」
2. 選擇「Service Account」方式
3. 輸入名稱：`GoogleSheets_專案一`
4. 開啟 Step 4.4 下載的 JSON 檔案，將內容**全部複製**貼到「Service Account Key」欄位
5. 點擊「Save」

【截圖位置：Google Sheets Credential 設定，Service Account 選項】

#### 8.3 設定要寫入的資料

**基本設定：**
- Document：選擇「By ID」
- Sheet ID：貼上 Step 3.4 取得的試算表 ID
- Sheet Name：`Sheet1`（或你的試算表名稱）

【截圖位置：Sheet ID 設定位置】

**設定欄位對應（Columns）：**
點擊「Add Column」逐一設定：

| 欄位名稱 | 值 |
|---------|-----|
| 時間 | `{{ $now }}` |
| 使用者ID | `{{ $('LINE').item.json.source.userId }}` |
| 輸入內容 | `{{ $('LINE').item.json.message.text }}` |
| 平台 | `{{ $json.content.match(/📱 \*\*建議平台\*\*：(.+)/)?.[1] || '未指定' }}` |
| 風格1_正式 | `{{ $json.content.match(/【正式風格】\n([\s\S]+?)(?=\n\n【輕鬆風格】)/)?.[1]?.trim() }}` |
| 風格2_輕鬆 | `{{ $json.content.match(/【輕鬆風格】\n([\s\S]+?)(?=\n\n【趣味風格】)/)?.[1]?.trim() }}` |
| 風格3_趣味 | `{{ $json.content.match(/【趣味風格】\n([\s\S]+?)(?=\n\n#️⃣)/)?.[1]?.trim() }}` |
| Hashtag | `{{ $json.content.match(/#️⃣ \*\*建議Hashtag\*\*：\n([\s\S]+?)(?=\n\n⏰)/)?.[1]?.trim() }}` |
| 建議發文時間 | `{{ $json.content.match(/⏰ \*\*建議發文時間\*\*：\n(.+)/)?.[1] }}` |

> 💡 **注意**：這些 expression 是為了從 AI 回應中解析出各個欄位。實際上建議先用簡單方式測試，再進階優化。

【截圖位置：Columns 設定畫面，顯示多個欄位對應】

**簡化版設定（建議初學者先用這個）：**
如果不想處理複雜的正則表達式，可以先這樣設定：

| 欄位名稱 | 值 |
|---------|-----|
| 時間 | `{{ $now }}` |
| 使用者ID | `{{ $('LINE').item.json.source.userId }}` |
| 輸入內容 | `{{ $('LINE').item.json.message.text }}` |
| 平台 | `{{ $json.content }}` |

這樣會把 AI 的完整回應都存到「平台」欄位，之後再手動整理。

### Step 12：加入條件判斷

#### 12.1 只處理特定關鍵字
如果你只想在收到特定訊息時才處理：

1. 在 LINE Trigger 和 AI Agent 之間加入「IF」節點
2. 搜尋「IF」並添加
3. 設定條件：
   - Value 1：`{{ $json.message.text }}`
   - Operation：`Contains`
   - Value 2：`幫我寫`

這樣只有訊息包含「幫我寫」時才會觸發 AI。

## 第四部分：成果驗收

### 驗收檢查清單

- [ ] LINE Bot 可以正常接收訊息
- [ ] 發送訊息後，AI 會生成 3 種風格文案
- [ ] 回覆包含 Hashtag 和發文時間建議
- [ ] 資料會自動寫入 Google Sheets
- [ ] 回覆訊息會通知使用者已完成儲存
- [ ] Workflow 可以持續運作（啟動後不會當掉）

### 常見問題排除

#### 問題 1：LINE Bot 沒有回應
**檢查項目：**
1. 確認 Workflow 是「Active」狀態
2. 確認 LINE Channel Access Token 正確
3. 確認 Webhook URL 設定正確（n8n 會自動設定）
4. 在 LINE Developers 的「Messaging API」頁面，確認 Webhook 是「Enabled」

【截圖位置：LINE Webhook 設定頁面】

#### 問題 2：AI 不回應或很慢
**檢查項目：**
1. 確認 OpenAI API Key 有效且有額度
2. 檢查 n8n 的 log 看有無錯誤訊息
3. 可能是網路問題，多等幾秒

#### 問題 3：Google Sheets 沒有寫入資料
**檢查項目：**
1. 確認 Service Account 有正確分享到試算表
2. 確認 Sheet ID 正確
3. 確認欄位名稱和試算表一致
4. 檢查 n8n 執行記錄（Executions）看錯誤訊息
